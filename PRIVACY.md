# Privacy Policy for Airbnb Insight AI
Last updated: March 2026

## **What This Extension Does:**
Airbnb Insight AI analyses guest reviews on Airbnb listing pages using OpenAI's API to help you make informed booking decisions.
Data We Collect and Process

1.  Data stored locally on your device:

Your OpenAI API key (required to use the extension)
Your travel preferences (optional, used to personalize analysis)
This data is stored using Chrome's local storage API and never leaves your device except as described below.

2. Data sent to OpenAI:

When you click "Analyze," the extension sends the following to OpenAI's API:

Review text from the current Airbnb listing
Listing title, location, and rating
Amenities listed on the page
Your travel preferences (if provided)

This data is sent directly to OpenAI using your personal API key. We do not operate any intermediate servers, and we do not have access to your API key or the data you send.

3. Data we do NOT collect:

We do not collect personal information
We do not track your browsing history
We do not use analytics or tracking services
We do not operate any servers that receive your data

Third-Party Services
This extension uses the OpenAI API. Your use of OpenAI is governed by OpenAI's Privacy Policy and Terms of Use.
Data Security

Your API key is stored locally using Chrome's secure storage API
All communication with OpenAI uses HTTPS encryption
No data is transmitted to any servers we operate

Your Rights
You can delete all stored data at any time by:

Removing your API key from the extension settings
Uninstalling the extension

Changes to This Policy
We may update this policy occasionally. Continued use of the extension constitutes acceptance of any changes.
Contact
For questions about this privacy policy, please open an issue on our GitHub repository.
